# Paper Tables

This directory contains LaTeX tables for the GroundCheck paper.

## Planned Tables

### Table 1: GroundingBench Statistics
- Category breakdown
- Number of examples per category
- Average memories, contradictions, difficulty

### Table 2: Overall Performance Comparison
- System comparison across all categories
- Overall accuracy, latency, cost

### Table 3: Contradiction Handling Detailed Results
- Focus on contradiction category
- Different contradiction types
- Resolution strategy accuracy

### Table 4: Ablation Study Results
- Impact of each component
- Timestamp vs trust resolution
- Disclosure verification impact

## File Format

Tables will be created in:
- Markdown (for review and editing)
- LaTeX (for paper submission)
- CSV (raw data)

## Notes

All tables will be generated from experimental results in `experiments/results/`.
