# Paper Figures

This directory contains figures for the GroundCheck paper.

## Planned Figures

### Figure 1: System Architecture
- Overview of GroundCheck's three-stage pipeline
- Fact extraction → Contradiction detection → Disclosure verification

### Figure 2: Contradiction Examples
- Visual comparison of how GroundCheck vs baselines handle contradictions
- Timeline showing temporal contradictions

### Figure 3: Performance Comparison
- Bar chart: Overall accuracy by system
- Grouped bar chart: Accuracy by category
- Focus on contradiction handling gap

### Figure 4: Error Analysis
- Confusion matrices for each system
- Breakdown of error types

### Figure 5: Latency vs Accuracy Trade-off
- Scatter plot: Accuracy vs latency
- Shows GroundCheck's efficiency advantage

## File Format

All figures should be provided in:
- PDF (vector format for LaTeX)
- PNG (raster format, 300 DPI minimum)
